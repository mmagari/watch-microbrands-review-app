# Watch Microbrands Review App

Small React project where users can explore independent watch microbrands and read or add reviews about them.

The idea of the project was to practice building a frontend application with filtering, sorting and a simple review system.

## What you can do in the app

* browse a list of microbrand watch companies
* search brands by name
* filter by style and price range
* sort brands by rating, name or price
* open a brand page with more details
* read reviews about the brand
* add your own review (saved in localStorage)

## Tech used

* React
* TypeScript
* Vite
* SCSS modules
* React Router

## How to run the project

Clone the repo and go to the frontend folder:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```

## Notes

Right now reviews are stored in **localStorage**.
In the future I would like to connect the project to a real database (for example Supabase).

The project is still a work in progress and I'm gradually improving the UI and functionality while learning more about frontend development.
